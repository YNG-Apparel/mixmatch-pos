# MixMatch POS

A web-based Point of Sale (POS) and inventory management system built with JavaScript and Supabase.

## Features

- Product management
- Inventory tracking
- Sales transactions
- Customer management
- Sales analytics
- Reports dashboard
- Role-based access control

## Technologies

- HTML5
- CSS3
- JavaScript
- Supabase
- PostgreSQL
- Netlify
